¿Quieres contribuir a LukaLibre?

LukaLibre es un proyecto 100% abierto. No necesitas saber programar para ayudar. Lo que más valoramos es tu tiempo, tus ideas y tus ganas de que más personas entiendan mejor sus finanzas.

👨‍💻 ¿Cómo puedes aportar?

  • Redacción de contenido: Ayúdanos a explicar conceptos financieros en lenguaje claro y chileno.
  • Corrección: Revisa textos, mejora la ortografía, aclara ideas.
  • Diseño: Mejora la experiencia visual, propone nuevas secciones o recursos.
  • Desarrollo: Contribuye al frontend o backend. Usamos HTML, CSS, Markdown, Python y FastAPI.
  • Difusión: Comparte LukaLibre con personas que puedan necesitarlo o colaborar.

🚧 ¿Qué estamos construyendo ahora?

  • Sitio web informativo y 100% responsivo.
  • Centro de conocimiento en GitHub Pages.
  • Calculadoras simples para sobrevivir mejor con tus lucas.
  • Simuladores de estafas y sobreendeudamiento.

🤝 ¿Dónde colaborar?

Todo el proyecto está en GitHub. Puedes crear un issue, proponer una idea o directamente hacer un fork y un pull request.

También puedes escribirnos a contacto@lukalibre.cl o simplemente difundir este sitio. Todo suma 🙌
